CentOS8下安装VMware® Workstation 15 Pro

# 安装依赖
sudo yum install elfutils-libelf-devel

一、下载VMware Workstation Pro 15.5.6 for Linux

点击前往下载页面下载安装文件

二、使用打算在运行 Workstation Pro 时使用的用户名登录主机系统

三、获得 root 用户权限

su root

四、将目录切换到包含 Workstation Pro 安装程序文件的目录

五、运行适合主机系统的 Workstation Pro 安装程序

# 使用 --console 选项运行安装程序，以便在终端中安装和配置 Workstation Pro
# 按照提示完成安装
sh VMware-Workstation-xxxx-xxxxxxx.architecture.bundle --console

六、启动VMware Workstation Pro

Workstation Pro 安装完成后， vmware-workstation-server 会在主机系统上启动。

每当重新启动主机系统时， vmware-workstation-server 都会启动。

Workstation Pro 启动时，请使用您的常规用户名（而非 root）登录。

直接点击图标或者打开控制台输入vmware后按回车键都会启动VMware Workstation Pro
